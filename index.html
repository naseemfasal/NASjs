Interactive Components

Try out all components right here. Click, interact, and see how easy they are to use.

📅

DatePicker

Beautiful date selection with month/year dropdowns, smart positioning, and min/max date restrictions.

NASjs.datePicker();
⏰

TimePicker

Intuitive time selection with hour/minute controls and AM/PM toggle. Supports 12 and 24 hour formats.

NASjs.timePicker();
🕐

Live Clock

Real-time clock display that updates every second with customizable format options.

NASjs.liveClock('#clock');
💬

Modal Popup

Minimal, beautiful modal with backdrop blur, smooth animations, and customizable buttons.

NASjs.modal(options);
✅

Confirmation

Beautiful confirm dialogs with custom callbacks for Yes/No actions and flexible styling.

NASjs.confirm(options);
💾

Basic Usage

<!-- HTML --> <input type="text" class="datepicker-input" placeholder="Select date"> <input type="text" class="timepicker-input" placeholder="Select time"> <div id="clock"></div> <script> // Initialize components NASjs.datePicker(); NASjs.timePicker(); NASjs.liveClock('#clock'); // Show modal NASjs.modal({ title: 'Hello!', content: '<p>Welcome to NASjs</p>' }); // Confirmation dialog NASjs.confirm({ title: 'Delete Item', message: 'Are you sure?', onConfirm: () => console.log('Deleted!'), onCancel: () => console.log('Cancelled') }); </script>
